+ Create new task
+ Update task to complete
+ Delete task
+ Support Responsive
## Usage
Run in local.
```bash
npm install
npm run dev
```
> [!NOTE]  
> This project using latest version of NextJS. Please install Node.js 18.17 or later.
> https://nextjs.org/docs/getting-started/installation
